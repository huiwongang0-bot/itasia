✅ 왜 Google Drive가 최적인가?
• 비용 없음 — Google Drive 15GB 무료 (업무용 서류 수백 건 저장 가능)
• 팀 공유 — 한국·이탈리아 사무실이 같은 파일에 동시 접근
• 크기 제한 없음 — 고화질 스캔본, 대용량 계약서 모두 OK
• 버전 관리 — 파일 수정 이력 자동 보관
• 모바일 접근 — 현장에서 스마트폰으로 바로 확인
📌 사용 방법 (3단계)
1
Google Drive에 폴더 구성
예: 직수입서류 → IMP-2024-001 → 계약서, B/L, 검역증...
2
파일 공유 링크 복사
파일 우클릭 → 공유 → 링크 있는 모든 사용자 → 링크 복사
3
이 시스템에 링크 등록
서류 관리 → 해당 수입건 → 서류 추가 → Drive 링크 붙여넣기
